# FHTH (For Happy To Happy)
반려동물 통합 관리 시스템


## Setup
```
$ git clone https://github.com/SSAFY5-free/FHTH_free.git
$ cd FHTH_free

// backend
$ cd /AWS/backend
$ npm install
$ npx nodemon

// frontend
$ cd AWS/frontend/fhth
$ npm install
$ npm run serve

// rpi backend 
$ cd backend_rpi
$ npm install
$ npm start
```

## Technical Stack
### Backend
- Node.js
- Express
- AWS (Amazon Web Service)
- MongoDB

### Frontend
- Vue.js
- Element-ui

### Local Server(이동형 CCTV)
- Linux
- Node.js
- Express
- Python

### ESP32 (급식기, 급수기)
- Arduino
- C/C++

### CI/CD
- Github
- Docker
- Nginx
