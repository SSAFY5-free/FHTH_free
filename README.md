## Setup
```
$ git clone https://github.com/SSAFY5-free/FHTH_free.git
$ cd FHTH_free

// backend
$ cd /AWS/backend
$ npm install
$ npx nodemon

// frontend
$ cd AWS/frontend/fhth
$ npm install
$ npm run serve

// rpi backend 
$ cd backend_rpi
$ npm install
$ npm start
```
